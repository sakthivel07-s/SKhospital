# SK Hospital Modern Web Platform

A premium, modern recreation of the SK Hospital landing page built with React, Vite, and Framer Motion.

## Features

- **Modern UI/UX**: Premium aesthetics with glassmorphism and smooth animations.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewports.
- **High-Performance**: Built with Vite and React for ultra-fast loading.
- **Animations**: Subtle entry and hover animations using Framer Motion.
- **Rich Visuals**: AI-generated high-resolution medical imagery.
- **SEO Optimized**: Semantic HTML and meta tags for better search visibility.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Modern Design System)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to the local URL (usually `http://localhost:5173` or `5174`).

## Project Structure

- `src/App.jsx`: Main application logic and components.
- `src/index.css`: Modern design system and core styles.
- `public/`: Assets like the hero image and tech background.
- `index.html`: Entry point and SEO configuration.
